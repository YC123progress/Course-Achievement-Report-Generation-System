# 课程目标达成评价报告生成工具

## 功能介绍

课程目标达成评价报告生成工具是基于若依管理系统开发的一个辅助工具，用于帮助教师根据学生成绩数据生成课程目标达成度评价报告。该工具可以：

1. 根据配置文件生成成绩录入模板
2. 通过在线Excel录入学生成绩数据
3. 自动计算课程目标达成度
4. 生成数据图表和评价报告

## 安装方法

本功能已集成到若依系统中，但需要在菜单管理中添加菜单项才能访问。详细安装步骤请参考 `INSTALL_MENU.md` 文件。

## 使用流程

1. **配置阶段**
   - 上传JSON格式的课程配置文件，或使用系统默认配置
   - 配置文件包含课程目标、考核方式、分值比例等信息

2. **数据录入**
   - 系统会根据配置生成Excel模板
   - 在在线Excel中录入学生成绩数据

3. **报告生成**
   - 点击"生成评价报告"按钮处理数据
   - 系统自动计算课程目标达成度并生成图表
   - 可下载完整Word格式评价报告

## 系统要求

- 若依管理系统V4.7.0+
- Python 3.6+（用于数据处理和图表生成）
- 所需Python库：pandas, numpy, matplotlib（已在requirements.txt中列出）

## 注意事项

1. 首次使用时，系统会自动安装所需Python依赖
2. 数据处理可能需要一定时间，请耐心等待
3. 生成的报告和数据将保存在系统中，可随时查看

如需技术支持或报告问题，请联系系统管理员。
